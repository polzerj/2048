# 2048
A simple 2048 tui game written in rust

## Installation
```bash
cargo install --git https://github.com/polzerj/2048
```

## Usage
```bash
tui_2048
```

## Controls
- `w` or $\uparrow$ - Up
- `s` or $\downarrow$ - Down
- `a` or $\leftarrow$ - Left
- `d` or $\rightarrow$ - Right
- `q` or `ESC` - Quit
